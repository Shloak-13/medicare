# Next.js Frontend

## What This Frontend Does

This frontend provides:

- login page
- JWT token storage for local development
- protected dashboard route
- healthcare dashboard shell
- sidebar navigation
- backend API connection to FastAPI

## Local Setup

Run these commands from:

```powershell
C:\Users\sapna shetty\Desktop\medicare\apps\web
```

Install dependencies:

```powershell
npm install
```

Create the local environment file:

```powershell
copy .env.example .env.local
```

Run the frontend:

```powershell
npm run dev
```

Open:

```text
http://localhost:3000
```
